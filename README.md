# Climate Change Analytics Dashboard

## Project Overview

Built a Tableau dashboard to analyze climate change indicators including temperature trends, air quality, precipitation, and extreme weather events.

## Tools Used

* Tableau
* Excel
* Data Analysis

## Key Insights

* Global Temperature Trends
* Air Quality Analysis
* Climate Risk Assessment
* Extreme Weather Event Analysis
* Environmental Data Visualization
